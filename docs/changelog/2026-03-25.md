# Changelog — 2026-03-25

## New Features

### Data Distribution Visual Mode (Pie Chart)

- Added a **Visual Mode** toolbar button next to the Dark/Light theme toggle; label reads "Visual" when in list mode and "List View" when in visual mode; button icon uses a colorful gradient pie symbol
- Toggling to Visual mode replaces the file list with a **donut pie chart** showing the top 10 largest items in the current folder
- An **"Other" slice** (muted gray) accounts for all items beyond the top 10
- **10 distinct colors** per slice: blue, rose, green, amber, violet, red, sky, orange, emerald, fuchsia
- **Legend panel** on the right of the chart shows each item's name, icon, size, and percentage of total folder size
- **Cross-highlighting:** hovering a pie slice highlights its legend row, and hovering a legend row highlights its pie slice
- Chart **auto-rebuilds** when async folder size calculations complete (uses `TaskContinuationOptions.OnlyOnRanToCompletion` to prevent stale rebuilds on cancelled tasks)
- Works identically in both **single-pane and split-pane** modes
- Pure WPF implementation — no third-party charting library; uses `PathGeometry` with `ArcSegment` (same pattern as the existing `UsageToArcConverter`)
- Uses `TryFindResource` for all brush lookups to remain safe across theme switches
- Event handler cleanup on each chart rebuild prevents listener leaks
