# Changelog — 2026-03-23

## New Features

### Bottom Pane Navigation Bar (Split Mode)

- Added a full-width navigation bar positioned between the top and bottom halves of split pane mode
- Includes Back, Forward, and Up buttons plus a read-only path display bound to `BottomCurrentPath`
- Full navigation history tracking for the bottom pane via `GoBackBottom`, `GoForwardBottom`, and `GoUpBottom` commands
- Layout is symmetric with the top navigation bar — spans across both the drive tree column and the file pane column
- **Layout restructured:** split pane mode now arranges as top half (source tree + file list), full-width bottom nav bar, bottom half (destination tree + file list), replacing the previous approach of separate column splits

### Deep Recursive File Search

- Search box in the top navigation bar now triggers a deep recursive file search when Enter is pressed or the magnifying glass button is clicked
- **Extension-aware matching:** typing `.exe` matches files by extension; typing `exe` matches by name substring
- Results populate inline in the top pane
- Manual recursive directory walking handles permission errors gracefully on a per-directory basis (inaccessible directories are silently skipped)
- "X" cancel button clears search results and returns to the normal folder view
- The existing live filter textbox continues to work as a secondary filter applied on top of search results

### Enhanced Refresh Button

- The Refresh button now reloads pane contents and recalculates folder sizes in addition to re-enumerating drives
- Handles edge cases gracefully: ejected drives and deleted folders no longer cause errors

## Bug Fixes

- **Search cancelled on tree navigation** — navigating via the drive tree now cancels any in-progress search, preventing garbled or stale results from appearing
- **CancellationTokenSource disposal** — tokens are now properly disposed after use to prevent resource leaks
- **Refresh with missing path** — Refresh no longer throws when the current path no longer exists (drive ejected, folder deleted)
